# TechAcademy Department Website

A responsive **College Department Website** built using **HTML and CSS**.
The project showcases academic programs, faculty members, department gallery, and contact information with a modern dark-mode design.

## Features

* Responsive multi-page website
* Modern dark mode UI
* Academic Programs page
* Faculty Directory (30 members)
* Department Gallery (30 images)
* Contact page with form
* Clean navigation between pages

## Pages

* Home (`index.html`)
* Programs (`programs.html`)
* Faculty (`faculty.html`)
* Gallery (`gallery.html`)
* Contact (`contact.html`)

## Technologies Used

* HTML5
* CSS3
* Responsive Layout
* GitHub Pages (for deployment)

## Project Purpose

This project was developed as a **B.Tech 1st Year Web Development Project** to demonstrate front-end development skills and build a modern college department website.
